A) Economy Rules (Do NOT change)
Daily token caps (across ALL games combined)

Free users: max 90 tokens/day

S-Class users: max 210 tokens/day

Rewarded games per day

Free: 3 rewarded games/day

S-Class: 6 rewarded games/day

Optional bonus: +1 rewarded game (if implemented)

Daily reset time

Reset occurs every day at 00:00 UTC

Core rules

Tokens can be earned only in Rewarded games

Practice games are unlimited but give 0 tokens

Rewarded games are a hard limit: if a user uses all rewarded games, they cannot earn more tokens that day even if they are below the daily token cap

If a game reward would exceed remaining daily allowance, deposit only the remaining amount and discard overflow

Tokens/runs not earned today are not saved and do not carry over

E) Game Page Rules (Player-facing text, simple language)

Add a visible “Token Rules” section or collapsible panel on the game page with this wording:

Free players can earn up to 90 tokens per day.

S-Class members can earn up to 210 tokens per day.

Tokens are earned only in Rewarded games.

Practice is unlimited but gives no tokens.

If you use all rewarded games, you can’t earn more tokens that day (even if you’re below the daily limit).

Tokens you don’t earn today are not saved for later.
Example: if a free player plays all 3 rewarded games and earns 65 tokens, only 65 tokens are added, even though the daily limit is 90.

Daily reset is 00:00 UTC.
